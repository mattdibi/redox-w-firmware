Openocd server

Install udev rules

sudo cp redox-w-firmware/49-stlinkv2.rules /etc/udev/rules.d/
udevadm control --reload-rules && udevadm trigger

Build the Openocd image

docker build -t redox-fw-openocd:latest --target redox-fw-openocd .

Run

docker run -p 4444:4444 --device /dev/bus/usb/002/012:/dev/bus/usb/002/012 -it redox-fw-openocd:latest
docker run -p 4444:4444 --device=/dev/bus -it redox-fw-openocd:latest

Toolchain server

Build the toolchain image

docker build -t redox-fw-toolchain:latest -target redox-fw-toolchain .

Run build and program

docker run --rm -v /home/mattia/Desktop/redox-w-firmware:/usr/src/nRF5_SDK_11/redox-w-firmware -dit redox-fw-toolchain:latest /bin/bash
